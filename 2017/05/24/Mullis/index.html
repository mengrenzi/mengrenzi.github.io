以下内容通过卖柠檬汁的有趣案例，帮助码农的我学会看懂财务报表以及学习会计相关知识：

资产负债表（Balance Sheet）

资产（左）：我们拥有的东西 == 负债（右）：这些东西属于谁

各表项简介：

现金：略
应收账款：略
存货：原材料 + 产成品
原材料：新鲜柠檬 + 白糖
产成品：柠檬汁
待摊费用：在未来会计期间有价值的费用（eg.insurance）
固定资产–建筑物：土地（土地不能折旧） + 建筑物
固定资产–设备：设备

应付账款：赊账购买原材料等物品而产生，还款期限短（30天左右），无利息
应付票据：收到他人的借款而产生，还款期限长（可能长达几年），有利息
应交税金：付给税务局的钱
负债合计：以上三项相加

初始投资：略
留存收益：以前的盈利，可留在“生意”中也可以分给公司股东
本周盈利：略
所有者权益合计：以上三项想加

相关概念

  • 费用：是指那些提出产品生产成本之外企业经营需要的花费，支出会使盈利减少
  • 人工成本：人工成本的支出会使现金减少，但不能简单费用化，人工成本增加了存货价值，存货变现，费用才体现
  • 收付实现制：除了依靠存货度量资产的权责发生制，还有依靠现金衡量的收付实现制，我们日常会计核算通常使用权责发生制。实行收付实现制仅仅是税务上的目的，一般卖产品的公司只能用权责发生制，服务业才能使用收付实现制。
  • 坏账：坏账是一项成本，减少是盈利。
  • 先进先出（FIFO）：物价上涨时，先进先出算法下，我们的商品销售成本更低，期末存货价值更高。
  • 后进先出（LIFO）：物价上涨时，后进先出算法下，我们的商品销售成本更高，期末存货价值更低。
  • FIFO与LIFO：出于税收的目的，盈利少，我们缴纳的所得税也相应减少，公司采用后进先出的目的是为了节税。存货在中国不能使用后进先出法。《企业会计准则第1号————存货2006》的规定
  • 固定资产不能费用化，因为这是一项价值较大且使用周期较长的支出。
  • 折旧：固定资产使用期间因耗用、磨损、过时导致的价值减少。折旧是一种非付现费用。
  • 加速折旧法：联邦通过鼓励公司买卖资产以刺激经济的方式。
  • 现金保证了公司的运作，利润是公司经营的最终目的。

期末资产负债表转期初资产负债表：

  1. 滚存利润，计入“留存收益”
  2. 将期末存货改为期初存货

利润表（P&L Statement）

实业：销售收入 - 销售成本 = 毛利（fat profit） - 费用 = 净利润（底线值）
服务业：营业收入 - 服务成本 = 毛利 - 费用 = 净利润

资产负债表和利润表联系

  1. 利润表将期初资产负债表和期末资产负债表相连接
  2. 利润表和资产负债表相同处：
    利润表.净利润 = 资产负债表.盈利
    利润表.期末存货 = 资产负债表.存货

现金流量表

现金流量表记录的是在一个给定期间内现金的流入和流出。

词汇表

accounting 会计 以货币为主要计量单位，采用一系列的方法和程序，对经济交易或事项进行连续、系统、综合的计量、反映及报告并从财务角度上，诠释了经营活动的各方面。

accounting equation 会计等式 资产 = 负债 + 所有者权益，会计核算应道遵循这一等式。

accrual 权责发生制 指会计核算中，已经济权利、责任为标准，确定本期收益和费用的一种制度。

capitalization 资本化将那些在一年内不完全消耗的成本计入固定资产的价值中。
