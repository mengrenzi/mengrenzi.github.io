Hexo常用命令

2017-05-22

平时经常用到的Hexo命令,简洁、易查。

清除缓存:

hexo clean

生成静态文件：

hexo g == hexo generate

启动服务预览（localhost:4000）：

hexo s == hexo server #启动服务预览

部署代码（上传到github）：

hexo d == hexo deploy#部署

新建文章：

hexo new “postName” #新建文章

新建页面：

hexo new page “pageName” #新建页面

Front-matter（用于指定个别文件的变量）：

参数 描述 默认值
layout 布局 post
title 标题 文件名
date 建立日期 文件建立日期
updated 更新日期 文件更新日期
tags 标签（不适用于分页）
categories 分类（不适用于分页）
thumbnail 缩略图地址
toc 显示 TOC 按钮 true
top 置顶 false
